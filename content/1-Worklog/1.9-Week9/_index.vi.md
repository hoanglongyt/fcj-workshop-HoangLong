### Mục tiêu tuần 9:

* Triển khai CI/CD lên Amazon EC2 bằng AWS CodePipeline.
* Quản lý EC2 thông qua IAM dựa trên Resource Tags.
* Nghiên cứu Grafana, CloudWatch Agent, AWS Systems Manager và AWS KMS.
* Phân tích nhật ký CloudTrail bằng Amazon Athena.

### Kết quả đạt được tuần 9:

* Hoàn thành triển khai CI/CD lên Amazon EC2 bằng AWS CodePipeline.
* Cấu hình IAM theo Resource Tags để kiểm soát quyền truy cập EC2.
* Tích hợp Grafana với Amazon CloudWatch phục vụ giám sát hệ thống.
* Sử dụng AWS Systems Manager quản lý Patch và thực thi lệnh từ xa.
* Cấu hình CloudWatch Agent thu thập chỉ số và nhật ký hệ thống.
* Phân tích nhật ký CloudTrail bằng Amazon Athena và kiểm tra các kịch bản phân quyền Decrypt với AWS KMS.
