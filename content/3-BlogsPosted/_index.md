This section lists and introduces the technical blog posts published to the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) community:

### [Blog 1 - SESSION POLICIES IN AMAZON EKS POD IDENTITY](3.1-Blog1/)
This blog introduces the new Session Policies feature in Amazon EKS Pod Identity, which allows narrowing IAM permissions flexibly and precisely for each Pod without creating separate IAM Roles, enforcing the Least Privilege principle in large-scale Kubernetes clusters.

### [Blog 2 - OPTIMIZING AUTOSCALING IN AMAZON EKS WITH KARPENTER](3.2-Blog2/)
This blog introduces Karpenter, a next-generation open-source Kubernetes Node Autoscaler built by AWS. Karpenter enables Just-in-Time node provisioning, pod consolidation, and EC2 Spot Instance optimization, reducing EKS cluster operational costs by up to 90%.

### [Blog 3 - AMAZON BEDROCK GUARDRAILS - ENTERPRISE SAFETY AND PRIVACY FOR GENERATIVE AI APPLICATIONS](3.3-Blog3/)
This blog introduces Amazon Bedrock Guardrails, providing multi-layered safety controls for Generative AI applications. It features denied topic filtering, PII redaction, and protection against prompt injection and jailbreak attacks.
